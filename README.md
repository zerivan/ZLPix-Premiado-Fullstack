# ZLPix-Premiado-Fullstack
🏆 ZLPix Premiado - Sistema de Sorteios com Pagamento via Pix (React + Vite + Node + Prisma)
---

## 🧰 Tecnologias

- React + Vite + TailwindCSS  
- Node.js + Express  
- Prisma ORM + SQLite  
- Render (Deploy automático)  
- GitHub Actions (CI/CD)

---

## ⚙️ Deploy no Render

1. Faça login em [https://render.com](https://render.com)
2. Conecte este repositório GitHub.
3. Render detectará automaticamente o `render.yaml`.
4. Clique em **Deploy**.
5. Acompanhe os logs e pronto 🚀
