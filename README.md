---

## 🚀 Deploy no Render

1. Acesse **[https://render.com](https://render.com)**
2. Conecte o repositório **ZLPix-Premiado-Fullstack**
3. O Render detectará automaticamente o arquivo `render.yaml`
4. Clique em **Deploy**
5. Pronto 🎉 Seu app estará online

```plaintext
