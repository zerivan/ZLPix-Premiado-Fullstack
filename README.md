# 🏆 ZLPix-Premiado-Fullstack

**ZLPix Premiado** — Sistema de sorteios com pagamento via Pix  
Desenvolvido em **React + Vite + Node.js + Prisma ORM**, totalmente integrado e hospedado no **Render**, com banco de dados **PostgreSQL (Neon)**.

---

## 🧰 Tecnologias Utilizadas

- ⚛️ **React + Vite + TailwindCSS** — Frontend rápido, moderno e responsivo  
- 🟢 **Node.js + Express** — API REST simples e escalável  
- 🧩 **Prisma ORM + PostgreSQL (Neon)** — Banco de dados relacional em nuvem  
- ☁️ **Render** — Deploy automático e CI/CD integrado  
- 🔐 **JWT + Bcrypt** — Autenticação segura de usuários  

---

## ⚙️ Deploy no Render

1. Acesse 👉 [https://render.com](https://render.com)
2. Conecte este repositório do **GitHub**
3. O Render detectará automaticamente o arquivo **`render.yaml`**
4. Clique em **Deploy**
5. Acompanhe os logs e aguarde o 🚀 **Deploy ser concluído**
